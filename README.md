# Calculator Package

This is a simple calculator package implemented in TypeScript, with CLI (Command Line Interface) support.

## Installation

To install the calculator package, you can use npm. Run the following command:

```bash
npm install my-calculator

Usage
You can use the calculator package by importing it into your TypeScript file and calling the appropriate functions. Here's an example:

import { welcome } from 'iamadeel-calculator';

welcome(); // This will prompt the user to enter numbers and an operator, and then perform the calculation.

Features
Addition
Subtraction
Multiplication
Division
Exponentiation
Example
Here's an example of how you can use the calculator package in your TypeScript file:

import { sum, subtract, multiply, division, power } from 'iamadeel-calculator';

sum(5, 3); // Output: 8
subtract(7, 2); // Output: 5
multiply(4, 6); // Output: 24
division(10, 2); // Output: 5
power(2, 3); // Output: 8
